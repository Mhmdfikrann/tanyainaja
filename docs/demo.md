Dokumen ini menyimpan screenshot dan alur demo singkat TanyainAja untuk reviewer, calon contributor, dan dokumentasi repository.

## Alur Demo Manual

1. Buka landing page di `/`.
2. Daftar atau login dengan WhatsApp OTP.
3. Masuk ke `/chat`.
4. Kirim pertanyaan pertama agar chat tersimpan di recent.
5. Upload gambar, lalu minta AI membaca isi gambar jika model aktif mendukung vision.
6. Buka halaman profile untuk melihat total request, token usage, ranking, dan top user.
7. Buka superadmin dashboard untuk memantau usage dari sisi admin.

## Catatan Demo

- Screenshot landing page tidak membutuhkan credential.
- Flow chat membutuhkan database, session, provider AI, dan konfigurasi OTP yang valid.
- Jangan menggunakan data user production untuk demo publik.
